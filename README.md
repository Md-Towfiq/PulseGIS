# PulseGIS
PulseGIS is a Python-based health monitoring tool for ArcGIS Enterprise that performs lightweight, payload-driven operational checks using web API requests. Unlike traditional machine-dependent tools, it enables scalable and remote Enterprise health validation.

# Usage Procedure
# Clone repository
git clone https://github.com/Md-Towfiq/PulseGIS.git

# Navigate to project
cd PulseGIS

# Install dependencies
pip install -r requests
pip install -r datetime

# Run health checks
python pulsegis.py
